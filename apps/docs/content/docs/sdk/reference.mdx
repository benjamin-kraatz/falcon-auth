# SDK Reference

## `createFalconAuthClient(config)`

Creates a new FALCON Auth client instance.

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `issuerUrl` | `string` | ‚úì | FALCON Auth server URL |
| `clientId` | `string` | ‚úì | Your OAuth client ID |
| `clientSecret` | `string` | ‚Äî | Client secret (server-side only) |
| `redirectUri` | `string` | ‚úì | OAuth callback URL |
| `scopes` | `string[]` | ‚Äî | Default scopes (default: `["openid", "email", "profile"]`) |

### Returns

`FalconAuthClient` instance.

---

## `client.getAuthorizationUrl(params?)`

Generates an OAuth authorization URL with PKCE.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `additionalScopes` | `string[]` | Extra scopes to request |
| `state` | `string` | Custom state (auto-generated if omitted) |
| `loginHint` | `string` | Pre-fill the email field |
| `prompt` | `"none" \| "login" \| "consent" \| "select_account"` | Force specific prompt |

### Returns

```ts
{
  url: string;          // Redirect users here
  state: string;        // Store this for CSRF verification
  codeVerifier: string; // Store this for code exchange
}
```

---

## `client.exchangeCode({ code, codeVerifier })`

Exchanges an authorization code for tokens.

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `code` | `string` | ‚úì | Authorization code from callback |
| `codeVerifier` | `string` | ‚úì | PKCE code verifier |

### Returns

```ts
{
  accessToken: string;
  tokenType: string;
  expiresIn?: number;
  refreshToken?: string;
  idToken?: string;
  scope?: string;
}
```

---

## `client.refreshTokens(refreshToken)`

Refreshes an expired access token.

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `refreshToken` | `string` | ‚úì | The refresh token |

### Returns

Same as `exchangeCode`.

---

## `client.getUserInfo(accessToken)`

Fetches user information from the userinfo endpoint.

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `accessToken` | `string` | ‚úì | Valid access token |

### Returns

```ts
{
  sub: string;
  name?: string;
  email?: string;
  emailVerified?: boolean;
  picture?: string;
  role?: string;
}
```

---

## `client.revokeToken(token, tokenType?)`

Revokes a token (signs the user out).

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `token` | `string` | ‚Äî | Token to revoke |
| `tokenType` | `"access_token" \| "refresh_token"` | `"access_token"` | Token type hint |

---

## `client.getDiscoveryDocument()`

Fetches the OIDC discovery document (`.well-known/openid-configuration`).

---
