[FALCON Connect](https://github.com/benjamin-kraatz/falcon-connect) is the workflow and automation layer in the FALCON ecosystem.

## Planned direction

- **Unified identity** — Connect workflows will be able to rely on FALCON Auth for user and organization context without duplicating directory data.
- **Event-friendly tokens** — OAuth scopes and service accounts will be aligned so Connect can call approved APIs on behalf of tenants.

Concrete integration points will ship in future milestones; this document will be updated as APIs stabilize.
